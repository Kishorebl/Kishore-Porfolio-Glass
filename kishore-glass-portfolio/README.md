# Kishore Glass Portfolio

Premium glassmorphism portfolio site for Kishore B L built with React, TypeScript, Vite, TailwindCSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

### Vercel
1. Push the repo to GitHub.
2. Import the project in Vercel.
3. Framework preset: **Vite**.
4. Build command: `npm run build`.
5. Output directory: `dist`.

### Netlify
1. Push the repo to GitHub.
2. Create a new Netlify site from the repo.
3. Build command: `npm run build`.
4. Publish directory: `dist`.

## Notes
